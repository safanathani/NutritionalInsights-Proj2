# Nutritional Insights Cloud App

Cloud-native web application built using React and AWS.

## Technologies
- React
- Amazon S3 (Static Website Hosting)
- AWS API Gateway
- AWS Lambda

## Features
- Nutritional insights dashboard
- Data visualizations (Bar, Scatter, Heatmap, Pie)
- Recipes and Clusters pages
- Pagination and filtering

## Live Application
http://nutritional-insights-frontend.s3-website.us-east-2.amazonaws.com
